Effortlessly build one page smooth scrolling using vanilla javascript

Over the years, I've always found the one page smooth scroll effect(demo) aesthetically pleasing and a joy to use when built well i.e not janky and responsive to user interaction. In the past, web developers have used various plugins and libraries to implement this effect and by doing so, they have come across challenges with regards to the difficulty of integrating these tools to their code base as well as adding extra bloat that could potentially affect the performance of their web apps. 

Because of these performance implications it is widely encouraged to either search for lightweight alternatives or even leverage on the API's and features modern JavaScript provides, making it simpler to write the functionality yourself.  Previously, writing your own logic to effectively produce the smooth scrolling effect was quite daunting, to get an idea of what it took, you can checkout Pete R's implementation or this blog post by Pawel Grzybek where he details his process of creating the effect. If you don't find the aforementioned code samples intimidating you can stop reading this now!  But before you do....
